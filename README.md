# MARK 利益判定ツール

電脳せどりの仕入れ判断を、売価と仕入値を入れるだけで **GO / 保留 / 見送り** に即判定するオフラインアプリ。

## 使い方
`rieki-hantei.html` をブラウザで開く（ダブルクリック）。ネット接続・インストール不要。

1. **Amazon売価**・**仕入値**・手数料・送料を入力
2. Keepaの **Amazon本体（不在/常駐）** と **先月の販売数** を入力
3. 右側に判定が自動表示

## 判定ロジック（MARKの鉄則）
| 判定 | 条件 |
|---|---|
| GO | 粗利 ≥ 目標 ＋ 仕入比 ≤ 0.6 ＋ Amazon本体不在 ＋ 回転あり |
| 保留 | 利益は出るが「本体常駐」or「低回転」の弱点あり |
| 見送り | 粗利が目標未満、または 仕入比 0.6 超 |

- 手残り = 売価 − 売価×手数料 − 送料・梱包
- 実質仕入 = 仕入 − 仕入×ポイント還元（楽天/Yahooせどり対応）
- 粗利 = 手残り − 実質仕入

「合格基準を調整」で 目標粗利・仕入比上限・最低回転 を変更可。ライト/ダーク・スマホ対応。

## 鉄則
セラーの相乗り売価を仕入れ基準にしない。現行定番は `仕入 ≤ 売価×0.6` が最低ライン。利益の源泉は店舗特価・卸・ポイント多重で作る。
